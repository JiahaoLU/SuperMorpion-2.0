the morpion game should be launched by following steps:
1. run 'pip install pygame' in terminal in both computers
2. on computer1 launch "Server_launch.py" until the game window pops up, note down the IP adress from the window
3. wait until on computer1 the game window pops up, on computer2 launch "Client_launch.py"
4. type in username on computer2, type in the IP adress gotten from computer1
5. play game on two computers under instructions on the bottom-right corner.
6. when one round is over, the one who has lost the game types 'r' to start a new game
7. type 'Esc' to quit the game
