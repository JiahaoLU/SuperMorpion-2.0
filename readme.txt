the morpion game should be launched by following steps:
1. run 'pip install pygame' in terminal in both computers
1. on computer1 launch "Server_launch.py" until the game window pops up
2. on computer2 launch "Client_launch.py"
3. type in username on computer2, type in the IP adress which is output from Server computer
4. play game on two computers under instructions on the bottom-right corner.
5. when one round is over, the one who has lost the game types 'r' to start a new game
6. type 'Esc' to quit the game
